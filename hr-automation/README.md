# HR Automation Workflow

## 📌 Project Overview
This workflow automates HR queries such as leave requests and policy questions using AI intent detection and Google Sheets.

---

## ⚙️ How the Workflow Works

1. Employee sends query via chat or webhook.
2. AI Agent detects intent (leave, policy, etc.).
3. HR data is fetched or updated in Google Sheets.
4. Conditional logic determines response.
5. Automated reply is sent to employee.

---

## 🛠 Key Nodes Used
- Chat Trigger / Webhook
- AI Agent (Intent Detection)
- Google Sheets
- Conditional Logic

---

## 💼 Business Value
- Reduces HR manual workload
- Instant employee response
- Consistent HR communication
