# Appointment Booking Automation

## 📌 Project Overview
This workflow automates appointment booking, rescheduling, and cancellation using AI Agent and Google Calendar integration.

---

## ⚙️ How the Workflow Works

1. User sends booking request.
2. AI Agent detects intent (book/reschedule/cancel).
3. Google Calendar availability is checked.
4. Event is created/updated/deleted.
5. Confirmation is sent back to user.

---

## 🛠 Key Nodes Used
- Webhook / Chat Trigger
- AI Agent
- Google Calendar
- Respond to Webhook

---

## 💼 Business Value
- Prevents scheduling conflicts
- Improves customer experience
- Automates appointment management
