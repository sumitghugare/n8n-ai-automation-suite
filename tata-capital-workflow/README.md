# Tata Capital Automation Workflow

## 📌 Project Overview
This workflow automates customer query handling using AI Agent and Google Sheets integration.  
It reduces manual processing and ensures faster response time.

---

## ⚙️ How the Workflow Works

1. Workflow starts via Webhook or Chat Trigger.
2. Incoming data is cleaned and validated using Code nodes.
3. AI Agent processes user intent based on Tata Capital business logic.
4. Required data is fetched or updated in Google Sheets.
5. Structured response is sent back to the requester.

---

## 🛠 Key Nodes Used
- Webhook / Chat Trigger
- Code Node (Data Formatting)
- AI Agent (LLM Integration)
- Google Sheets
- Respond to Webhook

---

## 💼 Business Value
- Reduces manual workload
- Ensures consistent responses
- Improves operational efficiency
