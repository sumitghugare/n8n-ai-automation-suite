# Instantly to Slack Automation

## 📌 Project Overview
This workflow fetches campaign data from Instantly API, summarizes it using AI, and posts updates automatically to Slack.

---

## ⚙️ How the Workflow Works

1. Triggered manually or via Webhook.
2. Instantly API fetches campaign data.
3. Code node formats the response.
4. AI Agent generates a human-readable summary.
5. Slack node posts the final message.

---

## 🛠 Key Nodes Used
- Webhook / Manual Trigger
- HTTP Request (Instantly API)
- Code Node
- AI Agent
- Slack Node

---

## 💼 Business Value
- Real-time campaign updates
- Eliminates manual reporting
- Improves team communication
